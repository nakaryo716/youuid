# YouUID
Build **your** own **UUID**

A universally unique identifier (UUID) is a 128-bit number
used to identify information in computer systems.

## Features
- [x] UUID version1
- [ ] UUID version3
- [ ] UUID version4
- [ ] UUID version6
- [ ] UUID version7
- [ ] UUID version8
